# Definition of Done (DoD)

> 一个Story被标记为"完成"前，必须同时满足以下7个条件。  
> 版本: v1.0 | 最后更新: 2026-06-02

---

## 七项硬性检查

| # | 检查项 | 验证方式 | 谁检查 |
|:--:|--------|---------|:------:|
| **1** | **代码完成** — 代码已编写、本地运行无报错 | 开发者本地启动验证 | 开发者 |
| **2** | **测试覆盖** — 相关单元/集成测试已添加并通过 | `mvn test` / `npx vitest run` | CI |
| **3** | **验收标准** — Story中所有AC全部勾选 | 逐条对照AC手动验证 | 开发者 |
| **4** | **无回归** — 现有测试套件全部通过 | CI Pipeline | CI |
| **5** | **代码审查** — P0/P1/安全相关Story需PR Review | GitHub PR Review Approved | Reviewer |
| **6** | **文档同步** — API变更已反映在Swagger | 访问 `/swagger-ui.html` 验证 | 开发者 |
| **7** | **部署验证** — 开发环境功能可正常使用 | 浏览器操作验证 | 开发者 |

---

## 按优先级差异化

| 优先级 | 必须满足 | 建议满足 |
|:------:|---------|---------|
| **P0** | 全部7项 | — |
| **P1** | #1, #2, #3, #4, #7 | #5, #6 |
| **P2** | #1, #3, #4, #7 | #2, #5, #6 |
| **P3** | #1, #4 | #3, #7 |

---

## Sprint完成定义

一个Sprint被标记为"完成"需要：

- [ ] 所有P0任务满足DoD
- [ ] P1任务完成率 ≥ 80%
- [ ] Sprint内无新增P0 BUG
- [ ] Sprint Retro已完成，行动项已记录
- [ ] `SPRINT-X.md` 已更新为最终状态
