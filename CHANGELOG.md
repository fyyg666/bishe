# 更新日志

本文档记录图书馆管理系统V2.0的所有版本变更。

## 版本规范

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) (语义化版本)：

- **主版本号 (MAJOR)**：破坏性API变更
- **次版本号 (MINOR)**：新增功能，向后兼容
- **修订号 (PATCH)**：问题修复，向后兼容

## v2.0.0 (2026-04-23)

> 全重构版本 - 基于原项目完全重写

### 新增功能

#### 认证模块
- **JWT双Token认证**：Access Token (2小时) + Refresh Token (7天)
- **账户锁定机制**：连续5次登录失败锁定15分钟
- **密码强度校验**：至少8位，包含大小写字母、数字、特殊字符
- **Token刷新接口**：支持无感刷新
- **账户禁用功能**：管理员可禁用账户

#### 图书管理
- **CRUD完整操作**：增删改查、批量导入
- **ISBN去重校验**：布隆过滤器加速查重
- **分类树形管理**：支持多级分类
- **热门图书推荐**：按借阅次数排序
- **Excel导入导出**：支持批量操作

#### 借阅管理
- **借书/还书/续借**：完整流程
- **借阅限制**：每人最多5本/30天
- **逾期罚款**：自动计算逾期费用
- **借阅统计**：多维度数据统计
- **分布式锁控制**：Redisson保证并发安全

#### 座位预约
- **阅览室管理**：多阅览室支持
- **可视化选座**：座位状态一目了然
- **签到签退**：时间精确控制
- **违约检测**：自动检测违规行为
- **预约冲突检测**：避免时间重叠

#### 信用积分
- **积分制度**：初始100分，等级制度
- **积分日志**：完整变动记录
- **积分规则**：加减分明细
- **积分查询**：实时余额查询
- **奖励兑换**：积分兑换礼品

#### 公告管理
- **公告发布**：富文本编辑
- **置顶功能**：重要公告优先展示
- **公告分类**：通知/活动/系统公告

#### 志愿服务
- **活动管理**：志愿服务项目管理
- **时长记录**：精确到小时
- **审核机制**：管理员审核
- **积分奖励**：审核通过增加积分

#### 统计分析
- **综合概览**：系统运营数据总览
- **借阅趋势**：历史数据图表
- **热门图书**：借阅排行
- **分类分布**：图书分类占比
- **月度统计**：周期数据对比

### 技术改进

#### 性能优化
- **二级缓存架构**：Caffeine L1 + Redis L2
- **分布式锁**：Redisson实现高性能锁
- **布隆过滤器**：快速ISBN查重
- **连接池优化**：HikariCP精细调优
- **异步处理**：减少阻塞操作

#### 安全加固
- **JWT认证**：无状态认证
- **密码加密**：BCrypt单向加密
- **XSS防护**：输入输出过滤
- **SQL注入防护**：参数化查询
- **API限流**：防止恶意请求
- **CORS配置**：限制跨域访问

#### 代码质量
- **自定义异常体系**：BusinessException等
- **全局错误处理**：统一响应格式
- **参数校验**：@Valid注解
- **Service分层**：Controller → Service → Mapper
- **常量类管理**：消除魔法值

#### 文档完善
- **API文档**：详细接口说明
- **数据库设计**：ER图和表结构
- **部署指南**：环境搭建和部署
- **开发指南**：前端开发规范

## 迁移指南

### 从 v1.0.0 迁移到 v2.0.0

1. **数据库迁移**
   ```sql
   -- 创建新数据库
   CREATE DATABASE library_system_v2 CHARACTER SET utf8mb4;
   
   -- 导入新schema
   source schema.sql;
   ```

2. **API变更**
   - 所有API路径已更新为 `/api/v1/`
   - 响应格式统一为 `{code, message, data}`
   - 需要JWT Token认证

3. **配置更新**
   - 更新application.yml配置
   - 配置新的数据库和Redis连接
