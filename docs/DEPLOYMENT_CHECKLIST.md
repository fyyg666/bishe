## 附录：快速检查脚本

### Linux/macOS

```bash
#!/bin/bash
echo "===== 快速环境检查 ====="
echo ""
echo "1. JDK: $(java -version 2>&1 | head -1)"
echo "2. Node: $(node -v)"
echo "3. Maven: $(mvn -v | head -1)"
echo "4. MySQL: $(mysql --version)"
echo "5. Redis: $(redis-cli --version)"
echo "6. Git: $(git --version)"
echo ""
echo "===== 服务状态 ====="
echo "MySQL: $(systemctl is-active mysql 2>/dev/null || echo 'N/A')"
echo "Redis: $(systemctl is-active redis 2>/dev/null || echo 'N/A')"
echo ""
echo "===== 健康检查 ====="
curl -s http://localhost:8080/api/v1/actuator/health | head -1
```

### Windows (PowerShell)

```powershell
Write-Host "===== 快速环境检查 ====="
Write-Host ""
Write-Host "1. JDK:"
java -version 2>&1 | Select-Object -First 1
Write-Host "2. Node: $(node -v)"
Write-Host "3. Maven:"
mvn -v | Select-Object -First 1
Write-Host ""
Write-Host "===== 服务状态 ====="
Get-Service -Name MySQL* | Select-Object Name, Status
Get-Service -Name Redis* | Select-Object Name, Status
Write-Host ""
Write-Host "===== 健康检查 ====="
try {
    Invoke-RestMethod -Uri http://localhost:8080/api/v1/actuator/health -UseBasicParsing
} catch {
    Write-Host "后端服务未运行"
}
```
